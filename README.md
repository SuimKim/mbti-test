### 3. 디렉토리 구조

```
📦src
 ┣ 📂assets
 ┃ ┗ 📂img        # 이미지 폴더
 ┣ 📂components   # 재사용 UI 컴포넌트
 ┣ 📂hook         # custom hook
 ┣ 📂page         # 각 페이지 컴포넌트
 ┣ 📂redux        # rtk - Slice와 Store
 ┣ 📂shared       # 페이지 이동을 위한 라우터
 ┣ 📂style        # 스타일 관련 컴포넌트
 ┣ 📜App.css
 ┣ 📜App.jsx
 ┣ 📜index.css
 ┣ 📜main.jsx
 ┣ 📜mockData.js  # MOCK DATA 파일
 ┗ 📜reset.css    # 기본 설정 css 초기화
```
